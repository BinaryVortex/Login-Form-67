# Login Form

A clean and modern login form built with HTML and CSS. This repository contains a simple, responsive login UI that uses Ionicons for icons and a blurred glassmorphism card over an underwater background image.

![Login Form Screenshot](./Screenshot%202024-07-29%20215622.png)

## Description

This project is a lightweight login form demo showcasing layout, input animations, and a glassmorphism effect. It is intended as a frontend-only example and does not include authentication logic or a backend — it is a UI mockup that can be integrated into larger projects.

## Features

- Simple, responsive login card centered on the page
- Floating labels on inputs
- Icon usage via Ionicons
- Glassmorphism effect with backdrop blur
- Background image included in the repository

## Files

- `index.html` — The HTML structure for the login form.
- `style.css` — Styling and layout using modern CSS (glassmorphism, floating labels).
- `Screenshot 2024-07-29 215622.png` — Screenshot used in this README.
- `view-underwater-sea-life-cartoon-style.jpg` — Background image used by the page.

## How to view

1. Clone the repository or download the ZIP.

   git clone https://github.com/BinaryVortex/Login-Form-67.git

2. Open `index.html` in your web browser. No build steps or server required. For the best results, open with a browser that supports backdrop-filter (Chrome, Edge, Safari).

## Customization

- Replace the background image (`view-underwater-sea-life-cartoon-style.jpg`) with your own to change the visual theme.
- Modify `style.css` to adjust colors, sizes, or animations.
- Add JavaScript and backend logic to perform real authentication if integrating into a full application.
